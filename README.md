# Homebrew Tap

Custom Homebrew formulas by [@subediparas5](https://github.com/subediparas5).

## Installation

```bash
brew tap subediparas5/tap
```

## Formulas

### lsport

A TUI for managing local and remote ports via SSH.

```bash
brew install subediparas5/tap/lsport
```

Or directly:

```bash
brew install subediparas5/tap/lsport
```
